# AMO Partners Chrome Extension

Простое расширение для упрощения работы с магазином партнера AMO. Работает локально в браузере, без синхронизации.

## Функциональность

- **Подпись ID-аккаунтов** - добавляйте понятные названия к числовым ID
- **Быстрый поиск** - находите аккаунты по ID или подписи во всплывающем окне
- **Визуализация в таблице** - сразу видите подписи прямо в таблице счетов

## Установка

1. Скачайте все файлы расширения в одну папку
2. Откройте Chrome и перейдите на страницу `chrome://extensions/`
3. Включите "Режим разработчика" (переключатель в правом верхнем углу)
4. Нажмите "Загрузить распакованное расширение"
5. Выберите папку с файлами расширения

## Использование

### Добавление подписи
1. Нажмите на иконку расширения
2. Введите ID аккаунта и подпись
3. Нажмите "Добавить"

### Поиск аккаунта
1. Введите ID или часть подписи в поле поиска
2. Нажмите "Найти" - расширение найдет и выделит аккаунт на странице

### Управление подписями
- Все подписанные аккаунты отображаются в списке
- Для удаления подписи нажмите × рядом с аккаунтом
- "Очистить все" удаляет все подписи

## Особенности

- Работает полностью локально
- Не требует синхронизации с сервером
- Автоматически обновляет подписи при изменениях на странице
- Поддерживает динамические страницы (SPA)

## Поддерживаемые страницы

- https://accounts.amocrm.ru/*
- https://www.amocrm.ru/*
